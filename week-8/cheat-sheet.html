JavaScript Objects quick guide

Basics

Objects can have properties and methods (or functions). All objects of the same type have the same properties but can have different property values. All objects of the same type have the same methods available to them but can be used at different times

Object Definition

Objects can be created using an Object literal

var dog = {name:"Charlie", age:7, breed: "corgi"};

Or by creating a new empty object and adding properties later

dog = new Object()

Access

Accessing properties

To access the properties of an object use either dot or bracket notation.

dog.name
--> Charlie
dog.["name"]
--> Charlie
Where dot notation accesses the property directly and anything in brackets is evaluated first.

Accessing methods

objectName.functionName();

Make sure to include the parentheses or it will return the function definition

Editing an Object

Adding Properties

To add a property to an existing object

dog.owner = "Barbara";

Adding Methods

dog.info = function () {
return this.name + " " + this.age;
};

Deleting Properties

Using the keyword delete

delete dog.age;

Looping

JavaScript uses the for ... in statement to loop through an object. The code block will be executed once for each Object property.

for (var in dog) {
console.log(dog[var]);
}
